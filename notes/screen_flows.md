# Screen Flows (IsmailNow)

- Flow: Create Habit
  - Habit Editor -> Save -> Habit Details
- Flow: Add Reminder to Habit
  - Habit Details -> Reminders -> Save -> Return to Habit Details
- Flow: Log Completion
  - Habit Details -> Mark Complete for Today -> Update HabitInstance + History
- Flow: View History & Analytics
  - Habit Details -> History & Analytics -> Date Range -> View Metrics
- Flow: Settings & Backups
  - Settings -> Backups -> Trigger Drive CSV export in Phase 4
