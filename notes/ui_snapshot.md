# UI Snapshots (IsmailNow)

- Habits Dashboard: overview screen with progress indicators; cards/list of habits
- Habit Editor: form for name, color, icon; cadence controls; notes; add reminders
- Habit Details: progress, history summary, quick actions (log completion, view history)
- Reminders: list of reminders per habit; add/edit time, repeats, timezone
- History: per-habit history list with date and completion state; quick stats
- Analytics: compact charts (trend line) and summary metrics (completion rate, streaks)
- Settings: backup toggle, locale, notification preferences
- Backup/Restore: UI to trigger Drive CSV backups in MVP rhythm
